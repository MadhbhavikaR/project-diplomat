# Demo Mode Environment

Create a local `.env.local` with the following values:

```
VITE_DEMO_MODE=true
VITE_API_URL=http://localhost:8080/api/v1
VITE_WS_URL=ws://localhost:8080/ws
VITE_REPO_API_BASE=/repo
VITE_AGENT_CONFIG_API_BASE=/agents
```

The `.env*` files are ignored in this repo, so this document serves as the template.
